# Проект «Закрывающий тег»

Личный блог-воспоминание о пути в веб-разработке

## 🎯 Цель проекта
Отработать навыки создания сложных декоративных элементов, анимаций и адаптивной верстки с использованием современных CSS-техник.

## 🛠 Технологии
- HTML
- CSS
- JavaScript (готовый код)

## ✨ Особенности
- Многослойные градиенты для фона
- Пиксельные SVG-иконки
- Сложные анимации с временными задержками
- Кастомные hover-эффекты с transform и transition
- Резиновая верстка через clamp()
- Вариативный шрифт Inter

## 🏗️ Структура
- Шапка с логотипом
- Карточки-воспоминания с изображениями и текстом
- Кнопки лайков с анимацией
- Модальное окно
